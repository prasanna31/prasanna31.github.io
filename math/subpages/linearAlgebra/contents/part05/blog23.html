Invertibility and Linear Isomorphisms

Invertibility in linear algebra is a central concept linking linear maps, matrices, and vector space structure. An invertible linear map is one that has an inverse map, allowing us to "undo" the transformation. Invertible maps are exactly the linear isomorphisms—linear transformations that establish a structural equivalence between vector spaces.

1. Definition of Invertible Linear Map

Let T: V → W be a linear map between vector spaces over the same field. T is called invertible if there exists a linear map T⁻¹: W → V such that:

T⁻¹ ∘ T = I_V,   T ∘ T⁻¹ = I_W

Where I_V and I_W are identity maps on V and W, respectively.

Intuition: Every vector in W is the image of exactly one vector in V, and we can recover v ∈ V from its image T(v) in W.

2. Linear Isomorphisms

A linear map T: V → W is an isomorphism if it is invertible. In other words:

  • T is linear
  • T is bijective (one-to-one and onto)
T is a linear isomorphism ⇔ T is invertible

Isomorphisms indicate that V and W are structurally equivalent. They have the same dimension and all linear properties are preserved under T.

3. Invertibility Criteria

A linear map T: V → V (square case) is invertible if any of the following equivalent conditions hold:

  • T is injective: Ker(T) = {0}
  • T is surjective: Im(T) = V
  • Matrix representation [T] relative to any basis is invertible (det([T]) ≠ 0)
  • Rank(T) = dim(V)

Note: For finite-dimensional spaces, injectivity, surjectivity, and invertibility are all equivalent.

4. Matrix Representation of Invertible Maps

If T: V → V is represented by a square matrix A = [T] in some basis, then:

T is invertible ⇔ A is invertible ⇔ det(A) ≠ 0

The inverse linear map T⁻¹ is represented by the matrix A⁻¹:

[T⁻¹] = [T]⁻¹

5. Example: Invertible Linear Map in ℝ²

Let T: ℝ² → ℝ² be T(x,y) = (2x + y, x + y)
Matrix representation in standard basis:
[T] = \begin{bmatrix} 2 & 1 \\ 1 & 1 \end{bmatrix}
Compute determinant: det([T]) = 2*1 - 1*1 = 1 ≠ 0 → T is invertible
Inverse matrix:
[T]⁻¹ = \begin{bmatrix} 1 & -1 \\ -1 & 2 \end{bmatrix}
Verify: [T]⁻¹ [T] = I

6. Geometric Intuition

  • Invertible linear maps preserve the "dimension" and "shape" of the vector space (up to stretching, rotation, reflection).
  • No collapse occurs: lines and planes are not squashed to lower dimensions.
  • The inverse map "reverses" these transformations exactly.

7. Properties of Invertible Linear Maps

  • Composition: If T: U → V and S: V → W are invertible, then S ∘ T: U → W is invertible, with (S ∘ T)⁻¹ = T⁻¹ ∘ S⁻¹
  • Inverse of inverse: (T⁻¹)⁻¹ = T
  • Identity map I is always invertible, and I⁻¹ = I
  • Preserves linear independence and span of vectors

8. Steps to Determine Invertibility

  1. Check dimensions: for V → W, dim(V) must equal dim(W)
  2. Compute kernel: T invertible ⇔ Ker(T) = {0}
  3. Compute matrix determinant (square case): det([T]) ≠ 0 → invertible
  4. If invertible, compute T⁻¹ by solving T(x) = y for basis vectors or using matrix inversion

9. Applications

  • Solving linear systems: invertible matrices correspond to unique solutions
  • Diagonalization and similarity transforms
  • Computer graphics: reversible transformations of images and coordinates
  • Quantum mechanics: unitary operators are invertible linear maps
  • Control theory: state transformations require invertibility for system equivalence

10. Summary Formulas

T invertible ⇔ ∃ T⁻¹ such that T⁻¹ ∘ T = I, T ∘ T⁻¹ = I
[T⁻¹] = [T]⁻¹ ⇔ det([T]) ≠ 0
Invertibility ⇔ T is bijective ⇔ linear isomorphism
(S ∘ T)⁻¹ = T⁻¹ ∘ S⁻¹

Invertibility ensures a perfect "one-to-one correspondence" between vector spaces and is exactly what makes a linear map a linear isomorphism, establishing structural equivalence between spaces.
