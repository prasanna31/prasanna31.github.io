Matrix Multiplication as Function Composition

Matrix multiplication is not just an algebraic operation—it is the concrete realization of function composition for linear transformations. Viewing matrices as functions allows us to see multiplication as "doing one linear transformation after another."

1. Linear Transformations and Matrices

Let T: U → V and S: V → W be linear maps between vector spaces U, V, W. Selecting bases for each space allows us to represent T and S as matrices:

  • [T] relative to bases of U and V
  • [S] relative to bases of V and W

These matrices encode exactly how basis vectors are mapped by the linear transformations.

2. Function Composition

The composition of two functions is defined as:

(S ∘ T)(u) = S(T(u)) ,   for u ∈ U

In words: first apply T, then apply S to the result. Composition preserves linearity if both T and S are linear.

3. Matrix Representation of Composition

If [T] and [S] are matrices representing T and S, then the matrix representing S ∘ T is the product:

[S ∘ T] = [S] [T]

Important: the order matters. First T, then S → multiply S on the left, T on the right.

4. Why Matrix Multiplication Works

Let T map basis vectors of U to V, and S map basis vectors of V to W. If we compute (S ∘ T)(u) for a vector u, we first express u in terms of U's basis, multiply by [T] to get coordinates in V, then multiply by [S] to get coordinates in W:

[u]_U → [T][u]_U = [T(u)]_V → [S][T(u)]_V = [S ∘ T](u)_W

Thus, matrix multiplication is exactly the application of one linear map after another in coordinate space.

5. Example: Composition in ℝ²

Let T: ℝ² → ℝ² be T(x,y) = (x + y, x - y)
S: ℝ² → ℝ² be S(u,v) = (2u, 3v)
Matrices in standard basis:
[T] = \begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix}, [S] = \begin{bmatrix} 2 & 0 \\ 0 & 3 \end{bmatrix}
Compute composition S ∘ T:
[S ∘ T] = [S][T] = \begin{bmatrix} 2 & 0 \\ 0 & 3 \end{bmatrix} \begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix} = \begin{bmatrix} 2 & 2 \\ 3 & -3 \end{bmatrix}
Check on vector v = (1,2):
  • T(v) = (1+2, 1-2) = (3, -1)
  • S(T(v)) = (2*3, 3*-1) = (6, -3)
  • Matrix multiplication: [S ∘ T] * [v] = [[2,2],[3,-3]] * [1,2]ᵀ = [6,-3] ✅

6. Geometric Intuition

  • Each matrix represents a linear transformation as stretching, rotating, shearing, or reflecting.
  • Matrix multiplication = applying one transformation, then the next.
  • The columns of the resulting matrix are exactly the images of the domain basis vectors under the composed transformation.

7. Properties of Matrix Multiplication as Composition

  • Associativity: (R ∘ S) ∘ T = R ∘ (S ∘ T) → (R S) T = R (S T)
  • Non-commutativity: Usually S ∘ T ≠ T ∘ S → matrix multiplication order matters.
  • Identity: The identity transformation corresponds to the identity matrix I, which acts as the neutral element for multiplication.
  • Inverse: If T is invertible, T⁻¹ corresponds to the inverse matrix [T]⁻¹, satisfying [T]⁻¹ [T] = I

8. Steps to Compute Matrix of Composition

  1. Express linear maps T: U → V and S: V → W in matrix form relative to chosen bases.
  2. Multiply matrices: [S ∘ T] = [S][T]
  3. Apply [S ∘ T] to any vector in coordinate form: [S ∘ T][v]_U
  4. Interpret geometrically: each column is where the basis vectors of U are mapped in W.

9. Applications

  • Computer graphics: successive transformations (rotate → scale → translate) are matrix multiplications.
  • Linear systems: combining transformations simplifies analysis of multiple operations.
  • Quantum mechanics: composition of linear operators represented as matrix multiplication.
  • Engineering: control systems, signal processing, where sequential operations are linear.

10. Summary Formulas

(S ∘ T)(v) = S(T(v))
[T] = matrix of T relative to basis
[S] = matrix of S relative to basis
[S ∘ T] = [S][T]
[T(v)] = [T][v]
[(S ∘ T)(v)] = [S][T][v]

Matrix multiplication is therefore exactly the algebraic manifestation of linear function composition, preserving linear structure and enabling computational efficiency.
