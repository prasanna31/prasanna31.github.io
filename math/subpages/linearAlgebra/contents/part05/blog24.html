Determinants — Algebraic and Geometric Meaning

The determinant is a fundamental concept in linear algebra that connects **matrix algebra** with **geometry**. It measures how a linear transformation scales space and provides crucial information about invertibility, volume scaling, orientation, and eigenvalues.

1. Definition of Determinant (Algebraic)

For a square matrix A = [aᵢⱼ] of order n×n, the determinant, denoted det(A) or |A|, is defined recursively:

  • For n = 1:
    det([a₁₁]) = a₁₁
  • For n = 2:
    A = [[a, b], [c, d]] → det(A) = ad - bc
  • For n ≥ 3 (Laplace expansion along row i):
det(A) = Σ (-1)^(i+j) aᵢⱼ det(Mᵢⱼ), i fixed, j = 1 to n

Where Mᵢⱼ is the (n-1)×(n-1) matrix obtained by removing row i and column j. This formula is recursive, expanding the determinant in terms of smaller determinants.

2. Properties of Determinants

  • det(I) = 1 (identity matrix)
  • det(Aᵀ) = det(A)
  • det(AB) = det(A) * det(B)
  • det(cA) = cⁿ * det(A) for scalar c and n×n matrix
  • Swapping two rows or columns multiplies the determinant by -1
  • If a row or column is zero, det(A) = 0
  • If a matrix has two identical rows, det(A) = 0

3. Determinant and Invertibility

A square matrix A is invertible ⇔ det(A) ≠ 0. Geometrically, det(A) = 0 means the transformation collapses space into a lower dimension (e.g., a line or plane), so no inverse exists.

4. Geometric Interpretation

Think of a linear transformation T: ℝⁿ → ℝⁿ represented by A. The determinant measures how T scales n-dimensional volume:

  • |det(A)| = scale factor of volume under T
  • det(A) > 0 → orientation preserved
  • det(A) < 0 → orientation reversed (like a mirror)
  • det(A) = 0 → volume collapses to lower dimension, not invertible

Examples:

2×2 case: T maps unit square to parallelogram spanned by columns of A = [[a, b], [c, d]]
Area of parallelogram = |det(A)| = |ad - bc|
3×3 case: T maps unit cube to parallelepiped spanned by column vectors of A = [[a₁, a₂, a₃], [b₁, b₂, b₃], [c₁, c₂, c₃]]
Volume = |det(A)|

5. Determinant via Row Operations

Determinants can be efficiently computed using row reduction (Gaussian elimination) with these rules:

  • Swapping rows multiplies determinant by -1
  • Multiplying a row by scalar c multiplies determinant by c
  • Adding a multiple of one row to another does not change determinant

After converting A to upper-triangular form U, det(A) = product of diagonal entries of U, adjusted for row swaps.

6. Examples

2×2 matrix: A = [[3, 2], [1, 4]]
det(A) = 3*4 - 2*1 = 12 - 2 = 10
Geometric meaning: T maps unit square to parallelogram of area 10
3×3 matrix: A = [[1,2,3],[0,1,4],[5,6,0]]
Compute det(A) using cofactor expansion along first row:
det(A) = 1*det([[1,4],[6,0]]) - 2*det([[0,4],[5,0]]) + 3*det([[0,1],[5,6]])
= 1*(1*0 - 4*6) - 2*(0*0 - 4*5) + 3*(0*6 - 1*5)
= 1*(-24) - 2*(-20) + 3*(-5) = -24 + 40 - 15 = 1
Volume of parallelepiped = 1 → unit volume preserved

7. Determinant as Orientation

If det(A) > 0, the orientation of basis vectors is preserved; if det(A) < 0, it is reversed. In 2D, this is like preserving clockwise/counterclockwise order; in 3D, it's like right-hand vs left-hand orientation.

8. Summary Formulas

det([[a]]) = a
det([[a, b],[c, d]]) = ad - bc
det(A) = Σ (-1)^(i+j) aᵢⱼ det(Mᵢⱼ) (Laplace expansion)
Volume scaling: vol(T(unit cube)) = |det(A)|
Invertibility: A invertible ⇔ det(A) ≠ 0

Determinants unify algebra and geometry, connecting matrices to volumes, orientations, and invertibility, making them essential in linear algebra, geometry, physics, and engineering.
