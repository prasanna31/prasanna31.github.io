Determinant-Based Proofs of Invertibility

Determinants provide a powerful algebraic and geometric tool for understanding when a square matrix or linear transformation is invertible. This page explores **why det(A) ≠ 0 is equivalent to invertibility**, and gives proofs, intuition, and examples.

1. Invertibility and Determinant: The Statement

Let A be an n×n matrix over a field F. Then:

A is invertible ⇔ det(A) ≠ 0

Equivalently, a linear map T: V → V represented by A is invertible ⇔ det(A) ≠ 0.

2. Intuition Behind Determinant and Invertibility

  • Determinant measures **volume scaling** under the linear map T.
  • If det(A) = 0, T collapses n-dimensional space into a lower dimension → vectors become linearly dependent → no unique inverse exists.
  • If det(A) ≠ 0, the map preserves n-dimensional volume → linearly independent → an inverse map exists that can "undo" T.

3. Proof Using Linear Independence

Let A = [a₁, ..., aₙ] be the column vectors of the matrix. Then det(A) ≠ 0 ⇔ {a₁, ..., aₙ} are linearly independent. Proof outline:

  1. Suppose det(A) ≠ 0. Then by definition, columns are linearly independent → equation A x = 0 has only the trivial solution x = 0.
  2. If columns are independent, A x = b has a unique solution for any b → A is invertible.
  3. Conversely, if det(A) = 0, columns are linearly dependent → A x = 0 has non-trivial solutions → not invertible.
det(A) ≠ 0 ⇔ columns of A are linearly independent ⇔ A invertible

4. Proof Using Row Reduction

Another approach uses **elementary row operations**:

  • If A can be reduced to identity matrix I using invertible row operations, then det(A) ≠ 0.
  • Row swaps multiply det by -1, scaling a row multiplies det by scalar, adding multiples of rows does not change det.
  • If det(A) ≠ 0, we can reduce A → I → construct A⁻¹ using same operations → invertible.

5. Proof Using Cofactor Expansion

For n×n matrix A, det(A) is sum over cofactors:

det(A) = Σ (-1)^(i+j) aᵢⱼ det(Mᵢⱼ)

If det(A) ≠ 0, at least one cofactor det(Mᵢⱼ) ≠ 0. This allows construction of **adjugate matrix** A⁺ such that:

A⁺ A = A A⁺ = det(A) I

Dividing by det(A) gives inverse:

A⁻¹ = (1 / det(A)) A⁺

This proves: det(A) ≠ 0 ⇒ A is invertible.

6. Geometric Perspective

Consider linear transformation T: ℝⁿ → ℝⁿ represented by A:

  • det(A) = scaling factor of n-dimensional volume under T
  • det(A) ≠ 0 → T preserves volume → vectors maintain independence → map reversible
  • det(A) = 0 → T collapses volume → vectors become dependent → no inverse
2×2 example: A = [[2,1],[1,1]]
det(A) = 2*1 - 1*1 = 1 ≠ 0 → invertible
3×3 example: B = [[1,2,3],[0,1,4],[5,6,0]]
det(B) = 1 → invertible

7. Summary Steps to Check Invertibility via Determinant

  1. Compute determinant of square matrix A.
  2. If det(A) ≠ 0 → A invertible, T is an isomorphism.
  3. If det(A) = 0 → A not invertible, linear transformation collapses space.
  4. Optional: compute inverse using adjugate or row operations if det(A) ≠ 0.

8. Summary Formulae

A invertible ⇔ det(A) ≠ 0
A⁻¹ = (1 / det(A)) adj(A) (if det(A) ≠ 0)
det(A B) = det(A) det(B) → composition of invertible maps is invertible

Determinants provide a rigorous and visual way to check invertibility: algebraically via formulas, geometrically via volume scaling, and computationally via row operations or cofactors. They unify structure, volume, and reversibility in linear algebra.
