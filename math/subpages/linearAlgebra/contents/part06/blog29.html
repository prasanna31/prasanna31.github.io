Jordan Canonical Form — Conceptual Understanding

The Jordan Canonical Form (JCF), also called the Jordan Normal Form, is a powerful conceptual tool in linear algebra that gives a complete understanding of a linear transformation or square matrix. Unlike diagonalization, which is possible only for matrices with enough independent eigenvectors, the Jordan form exists for every square matrix over an algebraically closed field (like ℂ).

1. The Idea Behind Jordan Canonical Form

Diagonalization works beautifully when a matrix has n independent eigenvectors. In this case, the matrix is "fully decomposable" into independent 1-dimensional invariant subspaces corresponding to each eigenvalue:

A = P D P⁻¹,   D = diag(λ₁, λ₂, ..., λₙ)

But what if there aren’t enough independent eigenvectors? That’s where the Jordan Canonical Form comes in. Conceptually, JCF is the closest possible “diagonal” form for any matrix. It organizes the matrix into blocks — called Jordan blocks — along the diagonal, where each block corresponds to an eigenvalue and encodes the presence of generalized eigenvectors.

2. Jordan Block: The Building Unit

A Jordan block for eigenvalue λ of size k is a k×k matrix:

J_k(λ) = \begin{bmatrix} λ & 1 & 0 & \cdots & 0 \\ 0 & λ & 1 & \cdots & 0 \\ \vdots & \vdots & \ddots & \ddots & \vdots \\ 0 & 0 & \cdots & λ & 1 \\ 0 & 0 & \cdots & 0 & λ \end{bmatrix}

Interpretation:

  • Diagonal entries = λ (the eigenvalue).
  • Superdiagonal = 1 (connects generalized eigenvectors).
  • Off-diagonal entries elsewhere = 0.

Geometrically, a Jordan block of size >1 represents an invariant subspace of dimension >1 where the matrix acts almost like a scalar (λ) but has a “shear” component connecting vectors in a chain. Each vector in the chain is a generalized eigenvector.

2.1 Generalized Eigenvectors

For a Jordan block J_k(λ):

  • The first vector v₁ is a regular eigenvector: (A - λI)v₁ = 0
  • The second vector v₂ is a generalized eigenvector: (A - λI)v₂ = v₁
  • In general, (A - λI)v_i = v_{i-1} for i = 2,...,k

This chain structure explains why diagonalization fails for matrices without enough independent eigenvectors. The Jordan form “records” the chain of dependence.

3. Conceptual Construction of JCF

Every square matrix A can be transformed via a similarity transformation to a block-diagonal form where each block is a Jordan block:

A = P J P⁻¹,   J = diag(J₁, J₂, ..., J_m)

Where each J_i is a Jordan block corresponding to an eigenvalue λ_i, and P is invertible. The key points:

  • The diagonal of J contains all eigenvalues (possibly repeated).
  • Each Jordan block represents one eigenvalue along with the structure of generalized eigenvectors.
  • The number of Jordan blocks for λ = geometric multiplicity of λ (number of independent eigenvectors).
  • The size of each block is ≤ algebraic multiplicity of λ.

3.1 Connection to Minimal Polynomial

The minimal polynomial directly determines the Jordan block sizes. If the minimal polynomial has a repeated factor (x-λ)^k, then the largest Jordan block for eigenvalue λ has size k. Repeated roots in the minimal polynomial correspond to chains of generalized eigenvectors, not just eigenvectors.

3.2 Connection to Characteristic Polynomial

The characteristic polynomial gives eigenvalues and their algebraic multiplicities. The Jordan form refines this information by showing the “block structure” within each multiplicity:

  • Algebraic multiplicity = sum of sizes of all Jordan blocks for λ
  • Geometric multiplicity = number of Jordan blocks for λ

4. Geometric Intuition

Think of a 2×2 matrix A with eigenvalue λ repeated twice but only one independent eigenvector:

A = [[λ,1],[0,λ]]

Action of A:

  • Vectors along the eigenvector direction get scaled by λ
  • Vectors in the generalized direction get scaled and “sheared” towards the eigenvector

JCF isolates these behaviors in blocks, showing exactly how the matrix stretches, scales, and shears along each invariant chain of vectors.

5. Why JCF is Conceptually Important

  • Every matrix has a Jordan form over ℂ → universal canonical form.
  • Encodes both eigenvalues and generalized eigenvector chains → structural insight.
  • Directly linked to minimal polynomial → block sizes reveal repeated roots.
  • Helps compute functions of matrices (e.g., exponentials, logarithms) using block-wise operations.
  • Illustrates why some matrices are diagonalizable and others are not.
  • Visualizes invariant subspaces: each Jordan block corresponds to a chain of nested invariant subspaces.

6. Examples (Conceptual)

1. Diagonalizable: A = [[3,0],[0,4]]
Jordan form: J = [[3,0],[0,4]] (already diagonal)
All eigenvectors are independent; no generalized eigenvectors.
2. Non-diagonalizable: A = [[2,1],[0,2]]
Jordan form: J = [[2,1],[0,2]]
One eigenvector (geometric multiplicity 1) and one generalized eigenvector forming a chain of length 2.
3. 3×3 matrix with repeated eigenvalue: A = [[5,1,0],[0,5,1],[0,0,5]]
Jordan form: J = [[5,1,0],[0,5,1],[0,0,5]]
Single eigenvector, chain of length 3 → illustrates largest Jordan block for λ=5.

7. Key Takeaways

  • JCF generalizes diagonalization to all square matrices.
  • Diagonalizable matrices → all Jordan blocks are size 1.
  • Non-diagonalizable matrices → Jordan blocks of size >1 capture generalized eigenvectors.
  • Jordan form visualizes both algebraic and geometric multiplicities of eigenvalues.
  • Minimal polynomial and characteristic polynomial dictate the Jordan block sizes and counts.
  • Understanding JCF is critical for advanced linear algebra, differential equations, dynamical systems, and matrix functions.

8. Summary Formulas and Concepts

A = P J P⁻¹,   J = diag(J₁, J₂, ..., J_m)
Each J_i = Jordan block of size k_i for eigenvalue λ_i
Algebraic multiplicity(λ) = sum of sizes of Jordan blocks for λ
Geometric multiplicity(λ) = number of Jordan blocks for λ
Minimal polynomial m_A(x) = Π (x-λ_i)^{k_i},   k_i = size of largest Jordan block for λ_i

In conclusion, the Jordan Canonical Form provides a conceptual lens to see exactly how a linear transformation acts on vector space: scaling along eigenvectors, shearing along generalized directions, and organizing the structure of invariant subspaces in a canonical, universal way.
