Span, Linear Combinations, and Generating Sets: A Deep Dive

Linear algebra is the study of vector spaces and the transformations between them. To become a true linear algebraist, one must deeply understand concepts such as span, linear combinations, and generating sets. These concepts form the backbone of vector space theory and appear in almost every topic in mathematics, physics, computer science, and engineering.

1. Linear Combinations

At its core, a linear combination is an expression constructed from a set of vectors using scalar multiplication and vector addition. Formally, if v_1, v_2, ..., v_n are vectors in a vector space V over a field F (e.g., real numbers ℝ or complex numbers ℂ), then a linear combination of these vectors is any vector v of the form:

v = a₁v₁ + a₂v₂ + ... + aₙvₙ

where a₁, a₂, ..., aₙ ∈ F are scalars. This simple definition is surprisingly powerful.

Example 1.1

Let v₁ = (1, 0) and v₂ = (0, 1) in ℝ². Any vector (x, y) in ℝ² can be written as:

(x, y) = x*(1, 0) + y*(0, 1)

Here, x and y are the scalars. Hence, (x, y) is a linear combination of v₁ and v₂.

Notice that linear combinations allow us to "build" new vectors from existing ones. The choice of scalars controls the direction and magnitude of the resulting vector.

1.1 Geometric Intuition

In two dimensions, linear combinations of a single vector v lie along the line through the origin in the direction of v. In three dimensions, combinations of two non-parallel vectors form a plane, while combinations of three vectors can fill the entire space (if they are linearly independent).

2. Span of a Set of Vectors

The span of a set of vectors is the collection of all possible linear combinations of those vectors. Formally:

Span{v₁, v₂, ..., vₙ} = { a₁v₁ + a₂v₂ + ... + aₙvₙ | a₁, a₂, ..., aₙ ∈ F }

In other words, the span of a set of vectors is the smallest vector space containing all of them.

Example 2.1

Consider v₁ = (1,0,0) and v₂ = (0,1,0) in ℝ³. The span is:

Span{v₁, v₂} = { (a, b, 0) | a, b ∈ ℝ }

Geometrically, this is the XY-plane in 3D space. Any vector in the XY-plane can be expressed as a linear combination of v₁ and v₂.

2.1 Properties of Span

  • Subspace: The span of any set of vectors is always a subspace of the vector space.
  • Minimality: Span is the smallest subspace containing the given vectors.
  • Closure: Span is closed under addition and scalar multiplication.

3. Generating Sets

A generating set (or spanning set) of a vector space V is a set of vectors whose span equals V:

V = Span{v₁, v₂, ..., vₙ}

That is, every vector in the space can be written as a linear combination of vectors from the generating set.

Example 3.1

In ℝ³, the standard basis vectors e₁=(1,0,0), e₂=(0,1,0), e₃=(0,0,1) form a generating set. Indeed, any vector (x, y, z) can be written as:

(x, y, z) = x*e₁ + y*e₂ + z*e₃

3.1 Minimal Generating Sets and Bases

While generating sets can be large or redundant, a basis is a minimal generating set. Formally, a basis is a set of vectors that:

  • Span the vector space
  • Are linearly independent (no vector can be expressed as a combination of the others)

The concept of a basis is central in linear algebra. A generating set that is not linearly independent contains redundancy; removing dependent vectors yields a basis.

Example 3.2

Consider ℝ² with vectors v₁=(1,0), v₂=(0,1), v₃=(1,1). The span of these three vectors is ℝ². However, v₁ and v₂ alone form a basis because v₃ can be expressed as v₁ + v₂. Thus, {v₁,v₂,v₃} is a generating set but not a basis.

4. Linear Dependence and Independence

Understanding generating sets requires knowledge of linear dependence. A set of vectors {v₁, ..., vₙ} is linearly dependent if there exist scalars a₁,...,aₙ, not all zero, such that:

a₁v₁ + a₂v₂ + ... + aₙvₙ = 0

If no such scalars exist (other than all zeros), the vectors are linearly independent. Independence ensures no redundancy in a generating set, which is exactly why bases are minimal generating sets.

Example 4.1

Vectors (1,0) and (0,1) are linearly independent in ℝ² because the only solution to:

a*(1,0) + b*(0,1) = (0,0)

is a=0, b=0.

5. Span and Geometric Interpretation

Geometrically, span tells us the "shape" formed by combining vectors:

  • One vector: Line through the origin
  • Two vectors: Plane if non-parallel, line if parallel
  • Three vectors in ℝ³: Volume of space if independent, plane or line if dependent

6. Examples Across Dimensions

Example 6.1: ℝ¹

Any nonzero vector spans the entire space. E.g., Span{2} = ℝ

Example 6.2: ℝ²

Vectors (1,0) and (0,1) span the plane. Adding (1,1) does not increase the span.

Example 6.3: ℝ³

Vectors (1,0,0), (0,1,0), (0,0,1) span the space. Any additional vector is a linear combination of these three.

7. Spanning Sets in Practice

Finding a spanning set is often the first step in understanding the structure of a vector space. Techniques include:

  • Using columns of a matrix to span the column space
  • Constructing bases for subspaces
  • Reducing a generating set to a basis using linear independence tests (e.g., Gaussian elimination)

8. Common Misconceptions

  • Span does not require the vectors to be independent.
  • Not every generating set is a basis.
  • A vector outside the span cannot be expressed as a linear combination of the set.

9. Exercises and Problems

To solidify understanding, try these exercises:

  1. Find the span of {(1,2), (3,4)} in ℝ². Are these vectors independent?
  2. Determine if {(1,0,1), (0,1,1), (1,1,2)} span ℝ³.
  3. Given vectors v₁=(1,0,0), v₂=(0,1,0), v₃=(1,1,0), find a minimal generating set.
  4. Explain geometrically what Span{(1,0,0), (2,0,0)} represents.
  5. Prove that the span of a set of vectors is always a subspace.

10. Summary

The concepts of linear combination, span, and generating sets are foundational in linear algebra. They help us answer questions like:

  • Which vectors can be built from a given set?
  • What is the minimal set needed to generate a vector space?
  • How do linear independence and dependence govern the structure of vector spaces?

Mastering these ideas is essential to advance into topics like bases, dimension, rank, eigenvectors, and vector space transformations. Geometric intuition combined with algebraic rigor is the key to truly internalizing these concepts. By carefully studying examples, experimenting with vectors in different dimensions, and solving exercises, one builds the skills to navigate the abstract landscape of linear algebra confidently.
