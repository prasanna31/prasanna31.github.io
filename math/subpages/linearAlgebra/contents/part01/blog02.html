Sets, Functions, and Mathematical Objects

The Language in Which All of Mathematics Thinks

1. Why Mathematics Begins Here

Every mathematical theory — linear algebra, calculus, probability, topology, category theory — rests on a silent foundation. That foundation is not numbers. It is not equations. It is not geometry.

It is the idea of objects, collections of objects, and relationships between objects.

Before we can speak of vectors, we must know what an object is. Before we can speak of transformations, we must know what a function is. Before we can speak of structure, we must know what it means to group things together.

This is why serious mathematics always begins with:

  • Sets
  • Functions
  • Abstract objects

This is not pedantry. This is epistemology.

---

2. Sets: The Act of Mathematical Attention

A set is often defined as:

A collection of distinct objects.

This definition is correct but shallow.

A deeper truth is:

A set is the result of deciding what matters and what does not.

When you form a set, you perform an act of abstraction. You ignore internal details and focus only on membership.

In this sense, a set is not a physical thing. It is a mental boundary.

Examples:

  • The set of real numbers ignores how numbers are written
  • The set of vectors ignores coordinate representation
  • The set of solutions ignores how they were found

Set theory is the mathematics of *classification*.

---

3. Membership: The Most Primitive Relation

The most fundamental symbol in mathematics is not + or =. It is:

∈ (belongs to)

To say x ∈ A is to assert a fact about reality — that an object falls within a chosen boundary.

Everything else is built on this relation.

Even equality is secondary.

Two objects are equal if they belong to the same sets in the same ways.

This is known as the principle of extensionality.

---

4. Subsets: Containment of Meaning

A subset is not merely a smaller set. It is a refinement of attention.

When we say:

A ⊆ B

we are saying:

Everything meaningful in A is already meaningful in B.

In linear algebra, subspaces are subsets with extra structure. In probability, events are subsets. In logic, propositions correspond to subsets of possible worlds.

Subset relations encode implication, restriction, and specialization.

---

5. Mathematical Objects: What Actually Exists?

A mathematical object is not a physical object. It has no mass. No location. No color.

A mathematical object exists by definition.

To define is to create.

Examples of mathematical objects:

  • Numbers
  • Functions
  • Vectors
  • Graphs
  • Groups
  • Spaces

Mathematics is not discovered in nature. It is constructed and then tested for consistency.

Once defined, objects obey rules — not opinions.

---

6. Functions: The Central Concept of Modern Mathematics

If sets describe what exists, functions describe how things relate.

A function is not a formula. It is not an expression.

A function is a rule that assigns exactly one output to each input.

This assignment is the essence.

The function exists independently of how it is written.

For example:

  • A lookup table is a function
  • A neural network is a function
  • A differential equation defines a function

Modern mathematics is function-centric. Objects are understood by how they transform under functions.

---

7. Domain, Codomain, and the Importance of Typing

Every function has:

  • A domain (where inputs come from)
  • A codomain (where outputs live)

This is not bookkeeping. This is logic.

The same rule with different domains is a different function.

Functions are typed objects.

Linear transformations are functions between vector spaces. Probability kernels are functions between measurable spaces.

Ignoring domains leads to conceptual errors.

---

8. Images, Preimages, and Information Flow

Functions move information.

The image of a set tells you what survives the transformation. The preimage tells you what causes an outcome.

In linear algebra:

  • Image → column space
  • Preimage → solution set

In ML:

  • Forward pass → image
  • Backprop → preimage sensitivity

Understanding functions is understanding causality.

---

9. Equality vs Isomorphism: A Mathematical Awakening

Beginners ask:

“Are these two things equal?”

Mathematicians ask:

“Are these two things the same in structure?”

Isomorphism is the idea that structure matters more than labels.

Two sets may be different, but if there exists a structure-preserving bijection, they are mathematically identical.

Mathematics studies structure up to isomorphism.

This idea prepares the mind for linear algebra, where bases change but spaces remain.

---

10. Why This Matters for Linear Algebra

Linear algebra is impossible without this foundation.

A vector space is a set with extra operations. A linear map is a function with extra constraints. A basis is a special subset.

Without sets, vectors are meaningless. Without functions, transformations do not exist. Without abstraction, structure cannot be seen.

---

11. The Mathematical Mindset You Are Building

After internalizing this chapter, you should:

  • Separate objects from representations
  • Respect domains and codomains
  • Think in terms of mappings, not formulas
  • Value structure over symbols

This is the mindset of:

  • Pure mathematicians
  • Theoretical physicists
  • ML theorists
  • Proof engineers
---

12. Closing Reflection

Sets teach you what exists. Functions teach you how things relate. Mathematical objects teach you how abstraction creates truth.

Once this foundation is solid, linear algebra becomes inevitable — not difficult.

You are no longer learning techniques. You are learning a way of seeing.
