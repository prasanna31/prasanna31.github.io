Finite vs Infinite Dimensional Vector Spaces

In linear algebra, the concept of dimension is fundamental. It captures the notion of the “degrees of freedom” in a vector space — essentially, how many independent directions exist in the space. Understanding the distinction between finite and infinite dimensions is critical for reasoning at the level of a linear algebraist. In this chapter, we explore this distinction rigorously, with proofs, examples, geometric intuition, and exercises.

1. Motivation and Concept of Dimension

Intuitively, dimension tells us how many vectors are needed to “describe” every vector in the space uniquely. For instance:

  • A line in ℝ² has dimension 1: any point can be described by one parameter along the line.
  • A plane in ℝ³ has dimension 2: any point can be expressed using two independent directions.
  • Three-dimensional space ℝ³ has dimension 3: three independent directions are needed.

Dimension is formalized using linear independence and span. A set of vectors is linearly independent if no vector in the set can be expressed as a linear combination of others. The span of a set is the collection of all possible linear combinations. The minimal set that is linearly independent and spans the space is called a basis, and its size defines the dimension.

2. Finite-Dimensional Vector Spaces

A vector space V over a field F is finite-dimensional if it has a finite basis. That is, there exists a finite set of vectors {v₁, v₂, ..., vₙ} such that:

  1. They are linearly independent.
  2. They span V.

The number of vectors in any basis is called the dimension of V, denoted dim(V). All bases have the same number of vectors — a fundamental property called the dimension invariance theorem.

Example 2.1: ℝ³

The standard basis of ℝ³ is e₁=(1,0,0), e₂=(0,1,0), e₃=(0,0,1). Any vector v=(x,y,z) can be written uniquely as v = x e₁ + y e₂ + z e₃. Hence dim(ℝ³) = 3.

Example 2.2: Polynomial Space P₂

P₂, the space of polynomials of degree ≤ 2, has basis {1, x, x²}. Any polynomial p(x) = a + bx + cx² is a unique linear combination of these basis vectors. Hence dim(P₂) = 3.

2.1 Properties

  • Any linearly independent set has at most dim(V) elements.
  • Any spanning set has at least dim(V) elements.
  • Every finite-dimensional vector space is isomorphic to Fⁿ.
  • Linear transformations can be represented as matrices.

3. Infinite-Dimensional Vector Spaces

A vector space V is infinite-dimensional if it has no finite basis. That is, for any finite set {v₁,...,vₙ}, there exists a vector v ∈ V not in their span.

Example 3.1: Space of All Polynomials P

The space P of all polynomials with real coefficients is infinite-dimensional. For any finite set {1, x, ..., xⁿ}, the polynomial xⁿ⁺¹ cannot be expressed as a linear combination of these. Hence, no finite basis exists.

Example 3.2: Continuous Functions C([0,1])

The space C([0,1]) of continuous functions on [0,1] is infinite-dimensional. No finite set of functions spans all continuous functions. For example, no finite combination of {1, x, x²} can represent sin(x).

3.1 Hamel and Schauder Bases

Even infinite-dimensional spaces have bases (Hamel bases) — linearly independent sets that span the space. Hamel bases are infinite. For practical purposes, Schauder bases allow infinite sums converging in a norm. Fourier series are examples of Schauder expansions in L² spaces.

Example 3.3: Hamel Basis of P

The set {1, x, x², x³,...} is a Hamel basis of P. Any polynomial can be expressed as a finite combination of these basis elements.

Example 3.4: Fourier Basis in L²([0,2π])

The orthonormal set {1, sin(nx), cos(nx) | n≥1} forms a Schauder basis of L²([0,2π]). Any square-integrable function can be represented as an infinite convergent sum of these basis elements.

Finite-dimensional spaces are visualizable:

  • ℝ²: plane
  • ℝ³: 3D space

Infinite-dimensional spaces can be conceptualized as “vector spaces of functions or sequences” where each coordinate corresponds to a basis element.

6. Coordinate Representations

Coordinates in finite dimensions are tuples of length dim(V). In infinite dimensions, coordinates with respect to Hamel bases exist but may be uncountably infinite. Schauder bases allow infinite sums converging in norm.

Example 6.1: Fourier Series

f(x) in L²([0,2π]) can be expressed as:

f(x) = a₀ + Σ (aₙ cos(nx) + bₙ sin(nx))

This demonstrates infinite-dimensional coordinates.

7. Applications

  • Quantum mechanics: wavefunctions in Hilbert spaces.
  • Signal processing: Fourier and wavelet analysis.
  • PDEs: solutions in function spaces.
  • Numerical analysis: finite-dimensional approximations of infinite-dimensional problems.

8. Exercises

  1. Prove that Pₙ is finite-dimensional and find a basis.
  2. Show that P is infinite-dimensional.
  3. Prove that C([0,1]) is infinite-dimensional.
  4. Explain why the Rank-Nullity theorem requires finite-dimensional spaces.
  5. Compute Fourier coefficients for f(x)=x on [0,π].

9. Summary

Finite-dimensional spaces allow full computation, geometric intuition, and theorem application. Infinite-dimensional spaces require subtle reasoning: bases are infinite, convergence matters, and coordinates may be infinite sums. Mastering this distinction is crucial for advanced linear algebra and functional analysis.
