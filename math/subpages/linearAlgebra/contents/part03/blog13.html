Exchange Lemma and Why All Bases Have the Same Size

The Exchange Lemma (also known as the Steinitz Exchange Lemma) is a cornerstone of linear algebra. It underpins the idea that the dimension of a vector space is an invariant, meaning that all bases of a finite-dimensional vector space contain the same number of vectors. This chapter will explore the lemma in depth, provide detailed proofs, multiple examples, geometric intuition, and exercises to fully internalize the concept.

1. Motivation and Historical Context

The question of whether all bases have the same size arises naturally when we consider vector spaces like ℝ² or ℝ³. For example, in ℝ², the standard basis {(1,0),(0,1)} seems “natural,” but other bases like {(1,1),(1,-1)} also exist. Are these fundamentally equivalent? Can we always measure the "size" of a vector space in terms of the number of independent directions? The Exchange Lemma, formulated in the early 20th century by Ernst Steinitz, provides a rigorous foundation to answer these questions.

Intuitively, the lemma formalizes the principle: independent vectors can replace vectors in a spanning set without losing the ability to span the space. This principle directly leads to the invariance of dimension.

2. Preliminary Concepts

Before stating the Exchange Lemma, let’s recall key concepts:

  • Linear Independence: A set S = {v₁, ..., vₘ} is independent if the only solution to a₁v₁ + ... + aₘvₘ = 0 is a₁=...=aₘ=0.
  • Spanning Set: A set T spans V if every v ∈ V can be expressed as a linear combination of vectors in T.
  • Basis: A set that is both independent and spans the space.

Example 2.1: ℝ³

Consider S = {(1,0,0),(0,1,0)} and T = {(1,1,0),(0,1,1),(1,0,1)}. S is independent, T spans ℝ³. The Exchange Lemma tells us that S can replace two vectors in T to form a new spanning set. This illustrates the general principle.

3. Statement of the Exchange Lemma

The Exchange Lemma: Let S = {v₁, ..., vₘ} be a linearly independent set in a vector space V, and T = {w₁, ..., wₙ} a spanning set. Then:

  1. There exists a subset of T of size m that can be replaced by the vectors in S to form a spanning set.
  2. Consequently, m ≤ n.

This lemma is fundamental because it quantifies the relationship between independent sets and spanning sets: an independent set can never be larger than a spanning set.

4. Proof of the Exchange Lemma

Step-by-Step Proof

We proceed by induction on the size m of the independent set S.

  1. Base Case (m=1): Let S = {v₁}. Since T spans V, v₁ can be expressed as a linear combination of vectors in T: 
    v₁ = a₁w₁ + ... + aₙwₙ, with some aⱼ ≠ 0
    Replace wⱼ with v₁. The new set spans V. Hence the lemma holds for m=1.
  2. Inductive Step: Assume lemma holds for m=k independent vectors. Let S = {v₁, ..., vₖ, vₖ₊₁} be independent.
    • By induction, {v₁, ..., vₖ} can replace k vectors in T to maintain a spanning set T'.
    • Now consider vₖ₊₁. Since T' spans V, express vₖ₊₁ as a combination of vectors in T': 
      vₖ₊₁ = b₁u₁ + ... + bₙuₙ, some bⱼ ≠ 0
      Replace uⱼ with vₖ₊₁. The set remains spanning.
  3. By induction, S can replace m vectors in T to form a spanning set. Therefore m ≤ n.

✅ Lemma proved.

5. Geometric Intuition

The Exchange Lemma can be visualized in low dimensions:

  • In ℝ², if you have two independent vectors forming a “diagonal” and a spanning set forming the standard basis, you can replace one of the standard vectors with a diagonal vector and still cover the plane.
  • In ℝ³, the principle generalizes: independent vectors define “directions,” and spanning vectors define a “volume.” Replacement preserves the volume while changing coordinates.

6. Why All Bases Have the Same Size

Theorem:

All bases of a finite-dimensional vector space have the same number of vectors.

  1. Let B₁ = {v₁, ..., vₘ} and B₂ = {w₁, ..., wₙ} be two bases.
  2. B₁ is independent and B₂ spans V. By the Exchange Lemma, m ≤ n.
  3. B₂ is independent and B₁ spans V. By the Exchange Lemma, n ≤ m.
  4. Thus m = n.

✅ This proves that dimension is invariant: the number of vectors in any basis is always the same.

Example 6.1: ℝ³

Standard basis: {(1,0,0),(0,1,0),(0,0,1)} (size 3). Diagonal basis: {(1,1,0),(0,1,1),(1,0,1)} (size 3). Despite different vectors, both bases have 3 vectors. Exchange Lemma guarantees this consistency.

7. Applications

  • Dimension Counting: Any independent set cannot exceed the dimension; any spanning set has at least the dimension of vectors.
  • Extending Independent Sets: Any independent set can be extended to a basis using the Exchange Lemma.
  • Rank of Matrices: Column rank = row rank; the lemma ensures consistency of counting independent columns/rows.
  • Subspace Analysis: Helps derive the dimension formula: dim(U + W) = dim(U) + dim(W) - dim(U ∩ W).

8. Exercises

  1. Apply the Exchange Lemma to S = {(1,0,0),(0,1,0)} and T = {(1,1,0),(0,1,1),(1,0,1)} in ℝ³. Replace elements of T to form a new spanning set.
  2. Prove that any independent set in a finite-dimensional space can be extended to a basis.
  3. Verify with an explicit example that all bases of ℝ² have 2 vectors.
  4. Consider a 4-dimensional vector space. Can an independent set have 5 vectors? Justify using the Exchange Lemma.
  5. Given U,W ⊆ ℝ⁴ with dim(U)=2, dim(W)=3, and dim(U∩W)=1, use the lemma to confirm dim(U+W)=4.

9. Summary

The Exchange Lemma formalizes the interplay between independent and spanning sets in vector spaces. It rigorously proves that all bases have the same size, explaining why dimension is invariant. Beyond finite-dimensional spaces, the principle underlies matrix rank, subspace decomposition, and extension of independent sets to bases. Understanding this lemma is crucial for any deep study of linear algebra.
