Climbing stairs

Given a staircase with n steps, you can climb 1 or 2 steps at a time.
Find the number of distinct ways to reach the top.

Approach: I want to take the steps and at a time I can take either 1 step or 2 steps. So, If I take one step at first, I have to count how many ways I can reach the top from the remaining n-1 steps. If I take two steps at first, I have to count how many ways I can reach the top from the remaining n-2 steps. So, the total number of ways to reach the top is the sum of the ways to reach from n-1 and n-2 steps. This gives us the recurrence relation: f(n) = f(n-1) + f(n-2) 


                func climbStairs(n int) int {
                    if n <= 0 {
                        return 0
                    }
                    if n == 1{
                        return 1
                    }
                    dp := make([]int, n+1)
                    dp[1], dp[2] = 1, 2
                    for i := 3; i <= n; i+=1 {
                        dp[i] = dp[i-1] + dp[i-2]
                    }
                    return dp[n]   
                }
